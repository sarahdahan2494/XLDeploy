This directory contains samples of XL Deploy customizations.

Source code of these samples can be found on Github:
    - https://github.com/xebialabs/xl-deploy-samples/

To install samples, copy jar-files into `plugins` directory of XL Deploy and restart the server.

To experiment with the samples and make changes to them, unpack the jar file and put the contents in the `ext` folder. Make sure to remove the jar-files.
