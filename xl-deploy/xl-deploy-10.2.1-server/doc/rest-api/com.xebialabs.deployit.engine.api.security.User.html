User

Users are rendered in XML as <user> elements.

Attributes

admin
Whether this is the 'admin' user (Boolean).

Child elements

<username>
The name of user.
<password>
The password of the user.

Example

<user admin="false">
  <username>deployer</username>
  <password>deployit</password>
</user>
