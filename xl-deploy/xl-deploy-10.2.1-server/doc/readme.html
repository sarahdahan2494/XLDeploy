Install XL Deploy 10.2.x

    Server requirements

    • XL Deploy license
    • Operating system: Microsoft Windows (32-bit or 64-bit) or a Unix-family operating system running Java
    • Oracle, IBM, or Apple Java Development Kit (JDK) 8
    • RAM: At least 2 GB of RAM available for XL Deploy
    • Hard disk space: Sufficient hard disk space to store the XL Deploy repository

    Depending on the environment, the following may also be required:

    • Database: XL Deploy's Jackrabbit repository supports a number of different databases
    • LDAP: To enable group-based security, an LDAP x.509 compliant registry is needed

    For networking requirements and information about computing hard disk space, visit https://docs.xebialabs.com.

    Client requirements

    GUI clients

    • Web browser:
      • Firefox
      • Chrome
      • Safari
    • Internet Explorer 10 or later
    • Flash player: Version 9 or later

    CLI clients

    • Operating system: Microsoft Windows or Unix-family operating system running Java
    • Oracle, IBM, or Apple Java Development Kit (JDK) 8

    Installation procedure

    To install XL Deploy:

    1. Download the XL Deploy ZIP file and license from https://dist.xebialabs.com.
    2. Extract the ZIP file to the directory where you want to install XL Deploy. This is referred to as <XLDEPLOY_HOME>.
    3. Copy the license file to <XLDEPLOY_HOME>/conf/xl-testview-license.lic.
    4. Open a command prompt and navigate to <XLDEPLOY_HOME>/bin.
    5. Execute one of the following commands to start the setup wizard:
      • On Unix-based systems: run.sh -setup
      • On Microsoft Windows: run.cmd -setup

    Follow the server setup wizard to install XL Deploy.
